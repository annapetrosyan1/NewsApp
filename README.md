iOS приложение для просмотра последних новостей с использованием [News API] (https://newsapi.org/).

Как запустить:
1. Клонируйте репозиторий
git clone https://github.com/annapetrosyan1/NewsApp.git
cd NewsApp

2. Откройте проект в Xcode

3. Зарегистрируйтесь на News API и получите бесплатный API ключ
   
4. Откройте NewsAppApp.swift и временно добавьте ключ:
    let tempKey = "ВАШ_API_КЛЮЧ_СЮДА"

5. Запустите приложение

После первого запуска ключ сохранится в Keychain и больше не будет храниться в коде!
